# Guia Turismo

Plataforma que **liga turistas a guias locais em Portugal**, com foco em **experiências personalizadas** de passeios turísticos — do formato de deslocação ao idioma e ao tamanho do grupo.

## O que oferecemos

Passeios guiados em diferentes formatos:

- **A pé**
- **Bicicleta**
- **Patinete**
- **Tuk tuk**
- **Carro**

Cada experiência é conduzida por **guias locais qualificados**.

## Como funciona

1. O cliente **escolhe o tipo de passeio**.
2. **Seleciona a data** e o **número de pessoas**.
3. **Escolhe o idioma** entre os disponíveis.
4. **Realiza a reserva** de forma simples.

Destaques para visitantes: **experiências personalizadas** e **variedade de tipos de passeio**.

## Para guias turísticos

- **Liberdade** para definir a própria agenda.
- **Possibilidade de aumentar a renda** com turismo.
- **Gestão simples** das reservas e atividades.

## Prova social

- **Avaliações reais** de clientes.
- **Experiências bem avaliadas**.
- **Guias verificados**.

## Stack

- [React](https://react.dev/)
- [Vite](https://vitejs.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/) (v4 com `@tailwindcss/vite`)

## Comandos

```bash
npm install
npm run dev
```

Servidor de desenvolvimento (por omissão em `http://localhost:5173`).

```bash
npm run build
npm run preview
```

Gera a pasta `dist/` e permite pré-visualizar o build de produção.
